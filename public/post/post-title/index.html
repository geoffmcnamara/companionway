Sorting IPs

Jan 1, 2017

I scan my home network frequently. There are a good number of smarthome devices, raspberry pies, phones, laptops, PCs etc. Using nmap to quickly scan a network is too slow for me - and it is a challenge to parse the output for easy reading (of course I would use awk to do that). But this listing is unsorted; I need it in ascending IP order. The solution is to creatively use the unix sort command declaring the field “termination” character as “.” and then declaring the key fields to sort on. Here is the final run:

$ sudo arp-scan -I wlo1 192.168.1.0/24|sort -t . -k3,3n -k4,4n
