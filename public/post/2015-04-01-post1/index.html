To get \(LaTex\) to work I had to add this to the _includes/footer.html You can actually add it right into the markdown file itself. I put it in the footer because it is a bit more efficient to have the js loaded last.

<script type="text/javascript"
    src="http://cdn.mathjax.org/mathjax/latest/MathJax.js?config=TeX-AMS-MML_HTMLorMML">
</script>

and change the _config to use redcarpet for markdown
